Introduccion

Python es un lenguaje de programación interpretado, de alto nivel y de propósito general. Fue creado por Guido van Rossum y se publicó por primera vez en 1991. Python tiene una sintaxis simple y clara que facilita la lectura y escritura del código. Además, Python admite múltiples paradigmas de programación, como el imperativo, el orientado a objetos, el funcional o el declarativo. Python también cuenta con una amplia biblioteca estándar que ofrece una gran variedad de módulos y funciones para realizar diversas tareas, como el procesamiento de texto, el manejo de archivos, las comunicaciones de red, el desarrollo web, el análisis de datos, la inteligencia artificial o el aprendizaje automático.

El programa más simple en Python es el que imprime el mensaje "Hola, mundo" en la pantalla. Para ello, solo hay que escribir la siguiente línea de código:

                
                    1. print("Hola, mundo")
Variables

En Python, una variable es un contenedor que almacena un valor. Para declarar una variable en Python, no es necesario especificar el tipo de dato. En cambio, Python determina automáticamente el tipo de dato almacenado en la variable.

Por ejemplo, para declarar una variable llamada a y asignarle el valor 5, simplemente escribiríamos:

                
                    1. a = 5

Luego, podemos imprimir el valor de la variable a en la consola utilizando la función print:

                
                    1. print(a)

En Python, existen varias maneras de asignar valores a variables, como por ejemplo:

                 
                    1. # Asignación directa
                    2. a = 5
                    3. # Asignación a través de una función
                    4. b = sum([1, 2, 3])
                    5. # Asignación a través de una operación aritmética
                    6. c = a + b
                    7. # Asignación a través de una operación lógica
                    8. d = True and False

Además, en Python se pueden utilizar números enteros con signos negativos y se pueden incluir números decimales en los flotantes. Por ejemplo:

                 
                
                    1. # Número entero positivo<
                    2. a = 5
                    3. # Número entero negativo
                    4. b = -10
                    5. # Número decimal
                    6. c = 3.14
                    7. # Número decimal negativo
                    8. d = -9.81
Tipos de datos

Los tipos de dato básicos en python son los siguientes:

  • Entero (int): Son números enteros sin signo, como 1, 2, 3, 1000, etc. Python admite enteros de tamaño arbitrario, solo limitados por la memoria disponible.
  • Flotante (float): Son números reales, que pueden tener una parte decimal. Los flotantes se utilizan para representar números reales que requieren una alta precisión. Por ejemplo, 3.14, 0.0001, 1000.1, etc.
  • Cadena de caracteres (str): Son secuencias de caracteres, como 'Hola', '1234567890', 'ABC', 'a string with spaces', etc. En Python, las cadenas de caracteres se pueden crear usando comillas simples ('') o dobles ("").
  • Lista (list): Son estructuras de datos que pueden contener varios valores. Las listas se pueden modificar después de crearse, lo que las hace flexibles. Por ejemplo, [1, 2, 3, 4, 5], ['apple', 'banana', 'cherry'], etc.
  • Tupla (tuple): Son estructuras de datos similares a las listas, pero inmutables. Es decir, no se pueden modificar después de crearse. Las tuplas se utilizan para almacenar datos que no cambiarán, como coordenadas (x, y, z), números de teléfono (código de país, número local), etc. Por ejemplo, (1, 2, 3, 4, 5), ('apple', 'banana', 'cherry'), etc.
  • Diccionario (dict): Son estructuras de datos que almacenan pares clave-valor. Son útiles para almacenar y organizar datos en pares clave-valor. Por ejemplo, {'nombre': '
Operadores

En Python, hay varios operadores que realizan operaciones diferentes. Algunos de los operadores más comunes en Python son:

  • Operadores aritméticos: Estos operadores realizan operaciones matemáticas simples, como suma, resta, multiplicación y división. Por ejemplo:
    •                  
                    1. a=5
                    2. b=3
                    3. mayor = a > b # mayor es igual a True
                    4. menor = a < b  # menor es igual a False 
                    5. igual = a == b  # igual es igual a False
                    6. desigual = a != b  # desigual es igual a True
  • Operadores lógicos: Estos operadores se utilizan para combinar valores booleanos y realizar operaciones lógicas, como AND, OR y NOT. Por ejemplo:
    •                  
                    1. a=True
                    2. b=False
                    3. and_operation = a and b #and_operacton es igual a False 
                    4. or_operacion = a or b  #or_operacion es igual a True
                    5. not_operacion = not a  # not_operacion es igual a False
  • Operadores de asignación: Estos operadores se utilizan para asignar valores a variables. Por ejemplo:
    •                  
                    1. a = 5
                    2. b = 3
                    3. a = a + b  # a es igual a 8
  • Operadores de comparación: Estos operadores se utilizan para comparar dos valores y devuelven un valor booleano (True o False). Por ejemplo:
    •                  
                    1. a = 5
                    2. b = 3
                    3. mayor = a > b  # mayor es igual a True
                    4. menor = a < b  # menor es igual a False
                    5. igual = a == b  # igual es igual a False
                    6. desigual = a != b  # desigual es igual a True
Estructuras de control

Las estructuras de control son formas en que el flujo del programa puede ser alterado por las decisiones.

Python ofrece varias estructuras de control que se utilizan para ejecutar bloques de código condicionalmente o repetidamente. A continuación, se detallan las principales estructuras de control en Python:

  • if: Se utiliza para ejecutar un bloque de código si una condición específica se cumple. La sintaxis es la siguiente:
    •                 
                    1. if condition:
                    2. # Bloque de código a ejecutar si la condición se cumple
  • elif: Se utiliza para agregar condiciones adicionales a una declaración if. El bloque de código asociado con elif se ejecutará si todas las condiciones previas sean falsas y la condición especificada en elif se cumpla. La sintaxis es la siguiente:
    •                 
                    1. if condition1:
                    2. # Bloque de código a ejecutar si la condición 1 se cumple
                    3. elif condition2:
                    4. # Bloque de código a ejecutar si condition1 no se cumple pero condition2 se cumple
  • else: Se utiliza para especificar un bloque de código que se ejecutará si ninguna de las condiciones anteriores se cumple. La sintaxis es la siguiente:
    •                 
                    1. if condition1:
                    2. # Bloque de código a ejecutar si la condición 1 se cumple
                    3. elif condition2:
                    4. # Bloque de código a ejecutar si condition1 no se cumple pero condition2 se cumple
                    5. else: 
                    6. # Bloque de código a ejecutar si ninguna de las condiciones anteriores se cumple
Funciones

Las funciones en Python son bloques de código que pueden ser ejecutados en cualquier parte del programa. Se definen mediante la palabra clave "def" seguida del nombre de la función y los parámetros que la función recibe, entre paréntesis.

Las funciones en Python son de dos tipos: funciones integradas, como print(), input(), type(), etc., y funciones definidas por el usuario, que se crean utilizando la palabra clave "def".

Algunas características importantes de las funciones en Python son:

  1. Parámetros: Son los valores que la función recibe al ser llamada. Los parámetros pueden ser de dos tipos: obligatorios y opcionales.
  2. Argumentos: Son los valores reales que se pasan a la función cuando se llama a la misma. Los argumentos se corresponden con los parámetros de la función.
  3. Valor de retorno: Es el valor que devuelve la función al finalizar su ejecución. Este valor puede ser utilizado por el código que llama a la función.
  4. Palabra clave "return": Esta palabra clave se utiliza para indicar que la función ha terminado su ejecución y que puede devolver un valor a la función que la llamó.

                    1. def suma(a, b):
                    2.     return a + b
                    3. 
                    4. resultado = suma(5, 10)
                    5. print(resultado) #Muestra: 15
